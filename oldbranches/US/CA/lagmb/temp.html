Title Bar

The section [in italics] is the only part of the page you need to change. Enter your text here and be sure to start your text with the "p" tag in "<" and ">" brackets at the beginning of each paragpagh. Be sure to end each paragraph with a "/p" tag in triangular brackets.

You may use whatever links you like; just remember to call them forward with your "a href" tags. Likewise you can use images when and where you want to with the "img src" tags.

You may use standard html code if you like (such as "h1", "b", and alternate fonts, but it is not recommended. This stylesheet is sufficient to provide a professional look.

Be sure to copy this template before you make changes to it so you have a cheat-sheet to refer back to.

To have red title bars use the 'p class="redbar"' tag (you must use a standard '/p' to close it). The tag for a grey index bar is 'p class="greybar"'. The 'a name=' tag allows you to anchor each red or grey bar so you can have a "jump to" link in the index section in the upper left-hand column. Just make sure your anchor names match your "a name" tags.
